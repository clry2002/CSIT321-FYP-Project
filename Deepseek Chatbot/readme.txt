1. Sign up for account at https://groq.com/
2. Gain an API key and name it whatever you want.
3. Place the API key in key = in the .env file.
4. Create an environment with with this in terminal. (python -m venv env)
5. Goto Terminal and ./env/scripts/activate
6. Afterwards, key in (pip install -r requirements.txt) to install Supabase and all the other AI Components.
7. In main.py, press F5 to run.
